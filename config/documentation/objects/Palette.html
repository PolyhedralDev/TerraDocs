Palette#

Represents a reference to a PALETTE config.

Palettes are defined as the referenced paletteâ€™s ID. For example a PALETTE config with an id of GRASS_MIX would be referenced in a palette parameter like so:

palette: GRASS_MIX

Uses#

Used by 2 parameters:

  • In base in BIOME:

    palette List<Map<Palette, Integer>>

  • In base in BIOME:

    ocean.palette Palette
