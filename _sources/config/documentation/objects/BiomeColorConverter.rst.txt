===================
BiomeColorConverter
===================

Types
=====

Different types of ``BiomeColorConverter`` provide different behaviours and may have additional parameters for configuring that behavior.

The type is specified by setting the ``type`` parameter to the name of the type. If the same name is used by two different addons, you can prefix the name with ``ADDON_NAME:`` to specify which one to use.

A list of available types for ``BiomeColorConverter`` are listed below:

---------

EXACT
-----

:superscript:`Added by the 'biome-provider-image-v2' addon`

`PARAMETERS:`

:bdg-primary:`match` :doc:`BiomeColorMapping` 

:bdg-primary:`else` :doc:`Biome` 

:bdg-success:`ignore-alpha` :doc:`Boolean` 

Default: ``true``
