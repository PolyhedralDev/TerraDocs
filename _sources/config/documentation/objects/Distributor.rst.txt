===========
Distributor
===========

Types
=====

Different types of ``Distributor`` provide different behaviours and may have additional parameters for configuring that behavior.

The type is specified by setting the :bdg-primary:`type` parameter to the name of the type. If the same name is used by two different addons, you can prefix the name with ``ADDON_NAME:`` to specify which one to use.

A list of available types for ``Distributor`` are listed below:

---------

SAMPLER
-------

:superscript:`*Type requires the 'config-distributors' addon to use`

:bdg-success:`threshold` :doc:`/config/documentation/objects/Float`\ 

Default: ``0.0``

:bdg-primary:`sampler` :doc:`/config/documentation/objects/NoiseSampler`\ 

---------

:superscript:`*Type requires the 'config-distributors' addon to use`
