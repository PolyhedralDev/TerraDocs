===============
GenerationStage
===============

Types
=====

Different types of ``GenerationStage`` provide different behaviours and may have additional parameters for configuring that behavior.

The type is specified by setting the :bdg-primary:`type` parameter to the name of the type. If the same name is used by two different addons, you can prefix the name with ``ADDON_NAME:`` to specify which one to use.

A list of available types for ``GenerationStage`` are listed below:

---------
