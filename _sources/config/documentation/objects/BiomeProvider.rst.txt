=============
BiomeProvider
=============

Types
=====

Different types of ``BiomeProvider`` provide different behaviours and may have additional parameters for configuring that behavior.

The type is specified by setting the :bdg-primary:`type` parameter to the name of the type. If the same name is used by two different addons, you can prefix the name with ``ADDON_NAME:`` to specify which one to use.

A list of available types for ``BiomeProvider`` are listed below:

---------

SINGLE
------

:superscript:`*Type requires the 'biome-provider-single' addon to use`

:bdg-primary:`biome` :doc:`/config/documentation/objects/Biome`\ 

---------

IMAGE
-----

:superscript:`*Type requires the 'biome-provider-image-v2' addon to use`

:bdg-primary:`color-conversion` :doc:`/config/documentation/objects/BiomeColorConverter`\ 

:bdg-primary:`color-sampler` :doc:`/config/documentation/objects/ColorSampler`\ 

:bdg-success:`resolution` :doc:`/config/documentation/objects/Integer`\ 

Default: ``1``

---------

PIPELINE
--------

:superscript:`*Type requires the 'biome-provider-pipeline-v2' addon to use`

:bdg-primary:`pipeline.source` :doc:`/config/documentation/objects/Source`\ 

:bdg-primary:`pipeline.stages` :doc:`/config/documentation/objects/List`\<:doc:`/config/documentation/objects/Stage`\> 

:bdg-success:`blend.amplitude` :doc:`/config/documentation/objects/Float`\ 

Default: ``0.0``

:bdg-success:`blend.sampler` :doc:`/config/documentation/objects/NoiseSampler`\ 

Default: ``Constant 0``

:bdg-success:`resolution` :doc:`/config/documentation/objects/Integer`\ 

Default: ``1``

---------

EXTRUSION
---------

:superscript:`*Type requires the 'biome-provider-extrusion' addon to use`

:bdg-primary:`extrusions` :doc:`/config/documentation/objects/List`\<:doc:`/config/documentation/objects/Extrusion`\> 

:bdg-primary:`provider` :doc:`/config/documentation/objects/BiomeProvider`\ 

:bdg-success:`resolution` :doc:`/config/documentation/objects/Integer`\ 

Default: ``4``
