=============
BiomeProvider
=============

Types
=====

Different types of ``BiomeProvider`` provide different behaviours and may have additional parameters for configuring that behavior.

The type is specified by setting the ``type`` parameter to the name of the type. If the same name is used by two different addons, you can prefix the name with ``ADDON_NAME:`` to specify which one to use.

A list of available types for ``BiomeProvider`` are listed below:

---------

SINGLE
------

:superscript:`Added by the 'biome-provider-single' addon`

`PARAMETERS:`

:bdg-primary:`biome` :doc:`Biome` 

---------

IMAGE
-----

:superscript:`Added by the 'biome-provider-image-v2' addon`

`PARAMETERS:`

:bdg-primary:`color-sampler` :doc:`ColorSampler` 

:bdg-primary:`color-conversion` :doc:`BiomeColorConverter` 

:bdg-success:`resolution` :doc:`Integer` 

Default: ``1``

---------

PIPELINE
--------

:superscript:`Added by the 'biome-provider-pipeline-v2' addon`

`PARAMETERS:`

:bdg-success:`resolution` :doc:`Integer` 

Default: ``1``

:bdg-success:`blend.sampler` :doc:`NoiseSampler` 

Default: ``Constant 0``

:bdg-success:`blend.amplitude` :doc:`Float` 

Default: ``0.0``

:bdg-primary:`pipeline.source` :doc:`Source` 

:bdg-primary:`pipeline.stages` :doc:`List`\<:doc:`Stage`\>
