=======
FEATURE
=======

:superscript:`*Config type requires the 'config-feature' addon to use`

This config type creates instances of :doc:`/config/documentation/objects/Feature`.

:bdg-primary:`extends` :doc:`/config/documentation/objects/List`\<:doc:`/config/documentation/objects/String`\> - A list of other ``FEATURE`` configs to copy parameters from.

Parameters from extended configs are only used if they have not already been defined in the current config. Configs listed first take precedence. This allows for re-use of parameters across multiple config files.

:bdg-primary:`id` :doc:`/config/documentation/objects/String`\ - An identifier used to reference this config from other configs.

The instance of :doc:`/config/documentation/objects/Feature` created by a config using this config type is identified using this parameter.
