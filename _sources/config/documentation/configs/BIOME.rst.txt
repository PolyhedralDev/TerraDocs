=====
BIOME
=====

:superscript:`*Config type requires the 'config-biome' addon to use`

This config type creates instances of :doc:`/config/documentation/objects/Biome`.

.. _config-biome-template-base-global-parameter-extends:

:bdg-ref-primary:`extends <config-biome-template-base-global-parameter-extends>` :doc:`/config/documentation/objects/List`\<:doc:`/config/documentation/objects/String`\> - A list of other ``BIOME`` configs to copy parameters from.

Parameters from extended configs are only used if they have not already been defined in the current config. Configs listed first take precedence. This allows for re-use of parameters across multiple config files.

.. _config-biome-template-base-global-parameter-id:

:bdg-ref-primary:`id <config-biome-template-base-global-parameter-id>` :doc:`/config/documentation/objects/String` - An identifier used to reference this config from other configs.

The instance of :doc:`/config/documentation/objects/Biome` created by a config using this config type is identified using this parameter.
